# WebDevComparison

**Comparison of different Web Development Stacks**

This project aims to compare various frontend technologies by implementing the **same three small applications** (World Clock, Timer, Countdown) using different tech stacks. This allows for a direct comparison of setup, complexity, developer experience, and performance.

## üì± The Apps

Each technology stack implements the following three apps with identical UX and features:

1.  **World Clock**: Display multiple timezones, add/remove clocks, highlight local time.
2.  **Timer**: Count up from zero, start/stop/reset, keyboard shortcuts.
3.  **Countdown**: Count down to a specific future time, visual notifications.
