# UI Guidelines

## Design System
*   **Colors**: Use a consistent color palette (e.g., dark mode by default).
*   **Typography**: Sans-serif font (Inter or system default).
*   **Spacing**: 8px grid system.

## Components
*   **Buttons**: Standard padding, rounded corners.
*   **Inputs**: Clear borders, focus states.
*   **Cards**: Used for containing apps (Clock, Timer, Countdown).
